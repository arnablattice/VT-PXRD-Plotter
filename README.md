# VT-PXRD-Plotter
A Python-based tool for visualizing and analyzing variable-temperature powder X-ray diffraction (VT-PXRD) data. Generates temperature-dependent 2Î¸ plots, heatmaps, and phase evolution profiles from stacked PXRD datasets.

# How users install & run (on their machine)
<br>
Clone the repo (or download ZIP).
<br>
Create a venv, then:
<br>
pip install -r requirements-dev.txt
<br>
pip install -e .
